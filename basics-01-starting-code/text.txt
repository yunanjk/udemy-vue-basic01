* methods vs computed vs watch

methods : 템플릿에서 데이터를 바인딩할 때
    ex) 데이터를 보간법에 사용하기 위해서
        템플릿에서 사용해야하는 데이터를 가져와야하기 때문
        이벤트 바인딩 위해서 


computed : 데이터 바인딩시에만 사용됨
            이벤트 바인딩시는 사용하지 않음
            의존하는 데이터가 변경될 때만 vue에 의해 재평가하거나 재실행됨
            

watch : 템플릿에 직접 사용하지 않음
        템플릿 내부에서 참조하지 않지만 연산 프로퍼티를 포함하여 어느 프로퍼티든 감시 가능

✅ 결과
이벤트 바인딩에서는 항상 methods(메서드) 사용
다른 데이터에 의존하는 데이터 출력에는 computed(연산 프로퍼티) 사용
